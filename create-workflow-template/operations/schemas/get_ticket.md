# `get_ticket` — Schema Reference

## Input Port: `input`

- **`id`** (`id`) **REQUIRED** — The id of the ticket.
  - ID type: `ticket`

- **`actual_close_date`** (`timestamp`) — Timestamp when the work was actually completed.
